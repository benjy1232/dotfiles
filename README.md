# Dotfiles
These are my dotfiles that have been getting use more recently so I have now seen a reason to start updating them.
For the most part not much has changed besides getting rid of things that were gonna be used for latex, which I never used.
I'm also going to start maintaining these config files on both github and gitlab in order to make sure I can get to them whenever I need to.

My current setup involves using Neovim with the following extensions
- [tpope/vim-pathogen](https://github.com/tpope/vim-pathogen)
- [OmniSharp/omnisharp-vim](https://github.com/OmniSharp/omnisharp-vim)
- [flazz/vim-colorschemes](https://github.com/flazz/vim-colorschemes)
- ayu-vim

If you're unfamiliar with any of those I have all of that information in .config/nvim/bundle besides ayu-vim.
Ayu is the current colorscheme set to dark mode and making with ```set termguicolors``` enabled to ensure that it looks the same despite color themes on the terminal itself.

The rest of the files in here are relatively old but still should be usable as I don't think bspwm or any of those programs have changed since the last time that I have used them.
Current font is IBM Plex Font, it can be found in the ```fonts``` folder or at 
[IBM's Website](https://www.ibm.com/plex/)
Note: All these fonts are in OTF format, if you want to use TTF, it is available at their website.
